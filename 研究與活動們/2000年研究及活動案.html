2000年研究及活動案

院務

  • 高雄願景二○二○ => 高雄市研究發展考核委員會 / 張清豐主持 高玉芬研究員。
  • 高雄市失業問題及其因應策略 =>高雄市研究發展考核委員會 / 張清豐主持、張維邦研究員。
  • 亞太企業與學術合作國際研討會 => 高雄大學主辦。
  • 本院與高雄大學、香港大學及大陸清華大學合辦「全球海港城市之轉變與新視野—全球海港城市城際網絡」研討會(2000.10.14~17)。
  • 跨世紀兩岸青年學生研習營(2000.09.21~30) 中華兩岸制度學會主辦，邀請大陸南京大學等五校師生來台訪問。

財經暨法律研究所

  • 中國大陸公司法上有關有限責任公司資本及營運問題之研究 =>趙東濟 撰。
  • 公務人力精簡策略之研究—以社教館所及社福機構委外模式研究為中心 => 行政院人事行政局 王仁宏主持 柯忠昇研究助理。

醫護研究所

  • 居家護理記錄品質之探討 => 杜敏世 楊清姿 研究。

科技研究所

  • 卡匣式電石英晶體研究 => 泰勝科技股份有限公司。
  • 光電科技在環保生態暨民生產業應用發展評估研究 => 國科會。
  • 金屬導線壓電石英晶體的研製 => 亞新科技公司。
  • 生物電子鼻電子構裝製程技術研發 => 大毅科技。
  • 陶瓷基座石英晶片研製 =>亞新科技公司。

人文暨社會研究所

  • 大陸泉州民間交流協會(華僑大學校長吳承業)等一行11人來台訪問，參加「台閩親緣文化交流學術研討會」(2000.01.23) => 本案獲文建會、海基會、中央陸工會及永達技術學院贊助。
  • 本院與高雄大學、永達技術學院組團赴大陸福建訪問，參觀廈門大學、華僑大學，並參加「兩岸高校與產學研合作」研討會及慶祝華僑大學四十週年(2000.10.28~11.02) => 本案獲陸委會中華發展基金補助。
