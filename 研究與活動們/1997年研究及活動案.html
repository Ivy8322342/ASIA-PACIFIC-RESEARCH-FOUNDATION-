1997年研究及活動案

院務

  • 香港城市發展參觀暨研討會(1997.03.05~09) 王創辦人等四十位代表參加，經由參觀考察之深度交流，吸取香港城市發展之經驗，探討兩岸三地未來城市發展規劃之可能性。 => 主辦單位：本院(洪美華副院長執行)、香港城市規劃及環境管理研究中心 地點：香港。
  • 海峽兩岸法律教育之現況與展望學術研討會(1997.03.29~31) 北京、南京、蘇州及杭州大學法律教授12人，假中山大學及永達工商專校舉行座談會 =>台大法律系、韓忠謨教授法學基金會主辦。
  • 馬來西亞衛生部長拿督蔡銳明一行23人來台訪問(1998.4.25~5.01) => 本院與順天堂藥廠(股)公司聯合邀請。
  • 一九九七後台灣-香港法律關係研討會(1997.5.16~18) => 地點：台大法學院。
  • 社會觀察與社會改革系列座談會實錄--一黑二黃三花四白五毒六法七騙(1997.1.25~4.17)七場 => 與王振生翁文教慈善基金會合辦。
  • 世界貿易組織架構下之貿易救濟制度研討會(1997.4.26) => 經濟部貿易調查委員會委託 (與中華亞太經濟與管理學會合辦)。
  • 認識APEC，發現新商機座談會 (1997.06.14及09.06) => 與台灣經濟研究院合辦。
  • 大連市臺灣事務辦公室左金夫先生率記者團等一行 => 參觀本院,晚宴招待。
  • 琉球沖繩國際大學產業總合研究所台灣考察團 => 參觀本院,晚宴招待。
  • 琉球自民黨婦女代表團 => 參觀本院及永達工商專校。
