1999年研究及活動案

院務

  • 韓國延世大學法學院院長朴吉俊來台參加「1999年東亞地區法學論壇(東亞地區經貿爭議解決機制)」學術研討會 (1999.03.17~23) => 台大法律系(王泰銓教授)、韓忠謨教授法學基金會主辦。
  • 工業園區與地方發展座談會 邀請蘇州市副市長謝家賓及蘇州工業園區等八人來台訪問，本院協助安排港務局行程 =>台大建築與城鄉研究發展基金會主辦 石上顧問公司執行。

財經暨法律研究所

  • 亞太地區動態經濟體之出口競爭力變化趨勢研究--顯示性比較利益指標與移動份額法之比較 =>行政院經濟建設委員會 /王鳳生主持。
  • 新加坡與香港的金融危機因應措施對我國發展金融中心的啟示 => 行政院經濟建設委員會 /王鳳生主持。
  • 公有土地信託制度與立法之擬議 => 行政院研究考核委員會 /王仁宏主持。
  • 宣導列車行動方案論文集(十五)-- 世界貿易組織(WTO)與亞太經合會(APEC)之反傾銷制度研討會(2000.01.22) => 經濟部貿易調查委員會 (羅昌發、蔡英文、徐純芳、洪德欽等教授主講)。
  • APEＣ貿易救論壇模式之建立 => 經濟部貿易調查委員會 王仁宏主持；王鳳生協同主持;陳英鈐、簡瑛助理研究。
  • 王仁宏教授演講文集 => 王仁宏。

醫護研究所

  • 護理之家住民特性分析 => 杜敏世 吳淑如 研究。
  • 居家護理病患照護需求之探討 => 杜敏世 楊清姿 研究 。

科技研究所

  • 壓電石英晶體之製作研究 => 泰勝科技公司委託。
  • 高精密度計頻器製作。

人文暨社會研究所

  • 「當前社會之道德重整與心靈改革—人文閞懷與文化再造」學術研討會(1999.01.04~05) -當前社會之道德重整與心靈改革論文集- => 本案獲教育部、文建會等六單位補助 杜英賢 編。
  • 海峽兩岸蘇雪林教授學術研討會 (於1999.08.20~27假大陸黃山盛大舉行) 蘇雪林先生學術研討會論文集出版 => 本案獲教育部、陸委會、海基會、文建會等補助。
  • 豐富人生關懷社會實踐研究案 => 行政院文化建設委員會。
