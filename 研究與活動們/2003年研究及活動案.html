2003年研究及活動案

院務

  • 各國貿易救濟相關制度及實務研究 => 經濟部貿委會/ 張清豐。
  • 王創辦人帶團赴大陸北京、天津、上海、蘇州、昆山等地各大學、機構參訪並舉行學術討論會 => 高雄大學 主辦。
  • 為紀念 故王天賞先生百歲冥誕，於11.16假高雄市歷史博物館舉辦「大高雄地區近百年文化變遷研討會」；並於11.07~20同地點展出「先賢手澤展」 => 本院及永達技術學院合辦；獲文建會等補助。
  • 邀請上海港口協會左端午副理事長等十三人來台進行參訪，並於12.18由本院主辦「兩岸港灣城市合作與互補研究」研討會，(2003.12.13~22) => 本案獲陸委會中華發展基金、高雄市政府港務局、高雄市都發局補助。
  • 受託辦理「2003高雄電影節」相關活動：記者會、國際電影觀摩展、郭南宏導演專題、嘉年華、動畫營等等；並於2004.01.01~ 01.03舉辦「國際論壇」邀請亞洲頂尖電影人做經驗上的交流 => 高雄市電影圖書館 /洪美華 本案獲新聞局、文建會、蒙委會…等機構補助。

財經暨法律研究所

  • 發展觀光條例暨相關法規翻譯 (就各觀光法規陸續進行翻譯) 。
  • 出版「我國電子支付暨信用工具法制導論」乙書 => 王仁宏教授指導編著/元照出版社經銷。

醫護研究所

  • 「左營、楠梓地區居家服務介入成效之探討」、 「影響日間照顧服務之利用相關因素之探討」 => 獎卿護理相關機構。

  • 「適合高雄未來發展的產業分析計畫」=> 高雄市研考會委託研究 /施明昌教授。
